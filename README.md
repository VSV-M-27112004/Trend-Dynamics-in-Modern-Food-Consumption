# Trend-Dynamics-in-Modern-Food-Consumption

Big Data Processing & Visualization using PySpark, Hadoop, and Kaggle Dataset( https://www.kaggle.com/datasets/shuyangli94/foodcom-recipes-with-search-terms-and-tags )

This project performs large-scale analysis on the Food.com Recipes Kaggle dataset using PySpark and Hadoop HDFS.
It includes complete data cleaning, transformation, feature engineering, and visual exploration of tags, ingredients, and recipe characteristics.
